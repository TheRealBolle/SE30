This folder contains jedec files to program a 16V8 device (GAL16V8, PALCE16V8, ATF16V8) as a replacement
for the video, clock and RAM-signalling PALs on the SE/30 logicboard.

Board locations and original part number reference:

UI6 341-0665-A
UH7 341S0689-C
UG6 341-0747-A
UG7 341-0746-A
UE6 341-0754-A
UE7 341-0744-A

The original PALs have been reverse-engineered and equations were rewritten to match the behaviour of the originals - the provided
JEDEC files are not dumps of the content of the original chips.
